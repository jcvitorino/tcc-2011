\chapter{Fundamentação Teórico-metodológica}

Em cada célula dos organismos vivos, para que ocorra a transcrição de um gene, é necessário a ação conjunta dos fatores de transcrição e dos elementos regulatórios. Os fatores de transcrição são proteínas que se ligam nos elementos regulatórios. Já os elementos regulatórios são pequenos segmentos de DNA localizados em uma região antes do gene que será transcrito. Essa região é chamada de região promotora (ou reguladora), na figura (colocar figura), podemos observar a região promotora e os elementos regulatórios, onde os fatores de transcrição se conectarão.
